# zhixing_CTAG2F90-C
driver of zhixing_CTAG2F90-C

# 介绍
CTAG2F90-C[35]是一款位置模式下的力矩速度控制高精度电动夹爪，采用伺服电机驱动，具有 90 mm 的最大夹持行程和可调节的夹持力，能够实现对不同材质和形状工件的柔性抓取。该夹爪支持标准工业通信接口，具备高重复定位精度，能够与主流协作机器人平台快速集成。凭借其紧凑的结构设计、智能控制功能以及高可靠性，CTAG2F90-C广泛应用于自动化装配、电子制造、医疗器械搬运及教育科研等领域，特别适合在要求高精度和柔性操作的应用环境中使用。
